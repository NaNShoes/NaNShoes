### 👨‍💻 Sobre Mí

Soy un **Desarrollador de Software Profesional** especializado en la creación de videojuegos, plataformas web y aplicaciones móviles. Me apasiona integrar tecnologías emergentes en mis proyectos, desde herramientas de **Inteligencia Artificial** para la generación de código hasta el ecosistema **Web3**, siempre manteniendo un fuerte enfoque en el diseño **UI/UX** para crear experiencias pulidas e inmersivas.

Actualmente colaboro con estudios de desarrollo (como gshelper.com) y gestiono mis propios proyectos interactivos.
